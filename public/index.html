This application is hosted on Heroku!

How to host on Heroku:

Step 1: Create application to host

Step 2: Create a new github repository

Step 3: Login to Heroku in CLI by using command "heroku login"

Step 4: Create new Heroku application by using command "heroku create enter_name"

Step 5: Upload application to github

Step 6: Link your desired github repository and bransch on your created Heroku application

Step 7: Deploy application on Heroku

Step 8: Done!
